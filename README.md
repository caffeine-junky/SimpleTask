# Simple Task

## Simple Task is a command line task management application
---

### Features
- Add Task
- Delete Task
- Update Task
- List All Tasks
- List Todo Tasks
- List In-Progress Tasks
- List Done Tasks
- Change Task Status
---

### How to use it
python <option> <value>

add "<description>"
delete <task_id>
update <task_id> "<description>"
list
list <status> | status being one of (todo, in-progress, done)
mark-in-progress <task_id>
mark-done <task-id>
